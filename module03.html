Introduction to CSS: Color, Text, Boxes, List, Tables & Forms

Summary of Content: In this module we started using CSS to style our webpages by linking our CSS file to our html file. When styling with CSS you can either put the link in the head tag, use the style tag in the header, or add the attribute style. You can change the color, text size, font, margins, and etc.
